# 基础

## 镜像源切换

### 1. 根目录创建 `.npmrc` 文件

```shell
registry=https://registry.npmmirror.com
```

### 2. yarn 配置修改

```shell
yarn config set registry https://registry.npmmirror.com
```

## npm 相关指令

### 登录

```shell
npm logout
```

### 包发布

```shell
npm publish
```

## 依赖新版发布淘宝镜像缓存处理
